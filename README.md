# Space Challenge App

Space Challenge App is Android Application for Training. Implemented in Kotlin. MVVM and Clean architecture used.

## General

Kotlin Language
Based on Clean + MVVM Structure

- Kotlin Version		: 1.6.0
- Min SDK Version	    : 21 (Android 5.0)
